# ruslamOS
![logo](image/exception.png)

my hobby 32bit os written in rust



## TOOLS

* Nasm
* Qemu
* Cargo (rust stuff)
* Make


# build
```
make
```

# run
```
make run
```

make sure your build system run on unix-like operating system
